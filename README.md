# Support Ticket System

A full-stack support ticket system where users can raise tickets and admins can manage and respond in real-time.

## Tech Stack
- React (Vite)
- Node.js & Express
- MongoDB Atlas
- Socket.IO
- Tailwind CSS

## Features
- User authentication (JWT)
- Single admin via env credentials
- Ticket creation & status tracking
- Admin reply & status update
- Real-time notifications (WebSocket)

## Setup
1. Clone repo
2. Install dependencies in `client` and `server`
3. Add `.env` in server
4. Run `npm run dev`
